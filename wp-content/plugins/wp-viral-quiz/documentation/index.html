Install

1. Go on your Wordpress Admin

 

2. Click "Plugins » Add new"

 

3. Click "Upload plugin"

 

4. Select the plugin zipped file and validate

 

5. Activate the plugin

Create a TrueFalse Quiz

1. Open WP Viral Quiz

 

2. Chose "Create a True/False Quiz"

 

3. Enter the name of your quiz

 

4. Create your first question 

5. Save the Quiz

 

6. Back on the WP Viral Quiz dashboard where your quiz is now present

 

7.  To insert a quiz in your article or page : copy and paste the shortcode to the content of your page/post, and save it ! The quiz will appear on the page.
